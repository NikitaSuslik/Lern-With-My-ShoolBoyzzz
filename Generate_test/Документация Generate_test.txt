При запуске, программа спросит куда вы хотите записать сгенерируемые тестовые данные.
0 - в папку с исполняемым файлом, 1 - прописать путь вручную.
Затем программа попросит ввести коэффициенты а и b и количество элементов.
После ввода всех данных произойдет генерация.
При успешной генерации и записи тестовых данных программа оповестит вас сообщениями:
"Генерация прошла успешно"
Второе сообщение будет содержать предложение пройти по пути, что бы ознакомиться с результатами генерации.
В результате генерации будут созданы два файла: "testData.test" и "testData.txt".
Первый файл подается на вход испытуемой программы "Uravnenie", второй файл содержит результаты генерации в человеческом понимании.
Если программа выведет "Ошибка при записи тестовых данных" - это значит, что произошла ошибка при записи одного из файлов, возможно у вас не хвататет прав для записи по заданному пути.Если вы используете автоматическое определение пути - попробуйте прописать путь вручную.
